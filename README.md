# Text Classification (AI Project)

## Overview
This project focuses on text classification using TensorFlow/Keras. It processes textual data and classifies it into predefined categories using a trained deep learning model.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy / Pandas

## Features
- Text preprocessing
- Model training and prediction
- Classification of input text

## How it Works
The model is trained on labeled text data. After preprocessing, it learns patterns and predicts categories for new input text.
